### Implementation av en ADT CirkulûÊr lista

Klassen ska namnges `CircularList` och implementera det generiska interfacet `ICircularList`

```csharp

{
    public void InsertAfterCurrent(T item);
    public void InsertBeforeCurrent(T item);
    public T GetCurrent();
    public void RemoveCurrentAndMoveForward();
    public void RemoveCurrentAndMoveBackward();
```

Med riktning framûËt avses att fûÑlja Next-pekaren och pûË motsvarande sûÊtt avser riktningen bakûËt att fûÑlja Prev-pekaren.

`GetCurrent` - returnerar aktuellt element.

`RemoveCurrentAndMoveBackward`- tar bort det element som current pekar pûË och ser till att current pekar pûË det borttagna elementets fûÑregûËngare.

`Contains` - avgûÑr om elementet finns i listan eller ej.

`MoveCurrentBackward` - flyttar current bakûËt det antal steg parametern nrOfSteps innehûËller.

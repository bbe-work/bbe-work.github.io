# Uppgifter
1. Vilken blir ordo-notationen för T(n) = nlogn + 2n + 34?

1. Vilken blir ordo-notationen för T(n) = 765

1. Ordna ordo-notationerna från snabbast till långsammast: O(n), O(n<sup>3</sup>), O(1) och O(nlogn)

1. Hur tolkar man O(f(n)) för en algoritm om f(n) = 1?

1. Hur tolkar man O(f(n)) för en algoritm om f(n) = n<sup>2</sup>?

1. Ge exempel på en algoritm som har tidskomplexiteten O(n).

1. Vilken är tidskomplexiteten T(n) för pseudokoden  

    ```
    i = 0
    end = n*n
    while i < end
        print(i)
        i = 1 + 1
    ```

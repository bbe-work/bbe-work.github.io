## Målet med kmom04

Efter att ha jobbat igenom kmom04 så ska du:

- förstå begreppet ADT
- kunna beskriva principerna för ADT:erna Stack och Kö

- kunna redogöra för tidskomplexiteten för respektive operation för ADT Stack och ADT Kö
- kunna redogöra för ADT:n Prioritetskö
